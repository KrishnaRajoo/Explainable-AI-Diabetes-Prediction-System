# Explainable AI Diabetes Risk Prediction System


## Project Overview

An AI-based diabetes risk prediction system using Machine Learning and Explainable AI.


## Technologies Used

- Python
- Streamlit
- XGBoost
- SHAP
- Power BI


## Features

- Diabetes risk prediction
- Risk probability
- SHAP explanation
- Patient report generation
- Interactive dashboard


## Machine Learning Model

Algorithm:

XGBoost Classifier


Performance:

Accuracy: 97.5%


## Dataset Features

- Age
- BMI
- Glucose
- HbA1c
- Blood Pressure
- Cholesterol
- Lifestyle factors
